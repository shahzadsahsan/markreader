Privacy Policy

Last updated: April 1, 2026

MarkScout is a local-only markdown file browser for macOS and iOS. Your privacy is fundamental to how we built this app.

What We Collect

Nothing. MarkScout does not collect, transmit, or store any personal data on external servers. There are no analytics, no tracking, no telemetry, and no advertising.

macOS App

Data Stored on Your Device

The macOS app stores the following data locally on your device:

  • App state — your favorites, reading history, filter preferences, zoom level, sidebar width, and scroll positions, stored in the app's sandboxed container directory.
  • Window state — window position and size so MarkScout reopens where you left it.
  • Watch directory paths — the folders you selected for MarkScout to scan, stored in the app container.

File Access

MarkScout reads markdown files from directories you explicitly select via the macOS file picker. The app has read-only access to your .md files and never modifies, deletes, or creates files in your watched directories.

iCloud Sync (Optional)

When iCloud sync is enabled in Settings, MarkScout copies your markdown files to a MarkScout folder in your iCloud Drive so the companion iOS app can read them. This is off by default. iCloud Drive sync is managed by Apple's operating system.

Network Access

The direct-download version of MarkScout makes a single HTTPS request to GitHub to check for updates. The Mac App Store version makes no network requests — Apple handles updates.

iOS App

Data Stored on Your Device

The iOS app stores the following data locally using standard iOS mechanisms:

  • Sync folder bookmark — a security-scoped bookmark to the iCloud Drive folder you selected, stored in UserDefaults.
  • Reading preferences — your selected theme, zoom level, and reading positions, stored in UserDefaults.
  • Cached files — copies of your markdown files in the app's Documents directory for offline reading. You can clear this cache from Settings.
  • Spotlight index — file titles and excerpts indexed via Core Spotlight so your files appear in system search.

iCloud Drive

MarkScout reads markdown files from a folder in your iCloud Drive that you explicitly select. The app has read-only access and never modifies, deletes, or uploads any files. iCloud Drive sync is managed by Apple's operating system.

Network Access

The iOS app makes no network requests. The only network activity is iOS downloading files from iCloud Drive, handled by the operating system.

Third-Party Services

MarkScout uses no third-party SDKs, analytics frameworks, crash reporters, or advertising networks on either platform.

Children's Privacy

MarkScout does not collect any data from anyone, including children under 13.

Changes to This Policy

If we ever change this policy, we'll update the date above and include the new policy in the next app update.
