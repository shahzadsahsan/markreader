## Description (4000 chars max)

Read your CLAUDE.md on the go.

MarkScout is a markdown reader built for developers who use AI-assisted coding tools like Claude Code, Cursor, and GitHub Copilot. Your project docs — CLAUDE.md, ARCHITECTURE.md, REQUIREMENTS.md — sync from your desktop to your iPhone via iCloud Drive, so you can review context, check constraints, and stay aligned with your codebase from anywhere.

**Why MarkScout exists:**
When you're building with AI, your markdown files ARE your project. CLAUDE.md defines how your AI assistant behaves. ARCHITECTURE.md captures every technical decision. REQUIREMENTS.md tracks what you're building and why. MarkScout gives you a beautiful way to read all of it from your phone — on the couch, on the train, or in a meeting.

**16 hand-tuned reading themes:**
Not just dark mode — choose from Obsidian, Deep Ocean, Synthwave, Tokyo Night, Catppuccin, Dracula, Monokai, Solarized, Nord Frost, Terminal Green, and more. Each palette is carefully tuned for comfortable reading of technical content, with distinct colors for headings, code, blockquotes, and tables.

**Built for developers:**
- Syntax-highlighted code blocks and formatted tables
- Table of contents with scroll rail for quick navigation
- Find-in-page search within any document
- Spotlight integration — your docs appear in system search
- Offline cache for reading without internet
- Pull-to-refresh for instant sync updates

**How it works:**
1. A companion desktop app syncs your markdown files to an iCloud Drive folder
2. Open MarkScout and pick that folder
3. Your files appear instantly — organized by project, sorted by time

Star your most-referenced files for quick access. Search across everything. Read offline. That's it — no accounts, no sign-up, no friction.

**Privacy-first:**
MarkScout collects zero data. No analytics, no tracking, no telemetry, no accounts. Your files stay on your device and in your iCloud Drive. We don't even make network requests — iCloud sync is handled entirely by iOS.

**Demo mode:**
Tap "Try Demo Mode" on the welcome screen to explore the full app with sample project files — no setup required.

## App Review Notes

**Demo credentials:** Not applicable — MarkScout does not require login.

**How to test without iCloud sync data:**
On the welcome screen, tap "Try Demo Mode" to load bundled sample markdown files. This lets you explore the full app experience — file list, reader view, favorites, search, themes, and settings — without needing a configured sync folder.

**How the app works:**
MarkScout reads markdown (.md) files from a user-selected folder in iCloud Drive. The user picks their sync folder during onboarding. The app is read-only and never modifies any files. A companion desktop application writes the markdown files and a manifest.json to the iCloud Drive folder, which MarkScout then reads.

Markdown files are commonly used by software developers for project documentation. This app is designed for developers who use AI coding assistants (like Claude Code, Cursor, or GitHub Copilot) that rely on markdown files such as CLAUDE.md for configuration and context.

**Required hardware/software:** iPhone running iOS 17.0 or later.
